Known Malfunctions

A common issue while using lcov arise when the gcov data does not match the latest binary files. If the source file is updated or rebuilt binaries, previously existing. gcno files or. gcda data files may not match the source code. And hence lcov run may indicate the files are corrupted or are mismatched as it uses the old. gcda and. gcno files.

So as a mitigation plan whenever the source file is updated and built, older. gcda and. gcno files should be removed before using lcov.
